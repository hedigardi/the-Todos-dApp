## Installation
1. Clone the Todos dApp repository:
   ```bash
   git clone https://github.com/hedigardi/the-Todos-dApp.git
    ```
2. Install the dependencies:
   ```bash
   npm install
    ```
3. Start the application:
   ```bash
   npm run dev
    ```
